TP sur le Falcon de François Lepan et Alexis Linke.

Pour ce TP nous avons implémenté une fonction permettant de créer du magnétisme vers une boule (Vec3D) passée en paramètre. 
La prochaine étape était d'ajouter plusieurs boules, cependant nous n'avons pas eu le temps de le tester.

Pour les n boules il suffit de rappeler la fonction avec un autre Vec3D.
